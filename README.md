# AI Desktop Companion 🧠

A desktop AI companion built with Python and PySide6, featuring LLM conversation, character personality system, animation interaction, and long-term memory management.

## ✨ Features

### 🤖 AI Conversation

- Support LLM-based conversation
- Asynchronous AI request with QThread
- Prevent GUI blocking during model inference
- Maintain multi-turn conversation context


### 🎭 Character Personality System

- Support multiple AI characters
- Independent character configuration
- Custom system prompts

Example:

- Paimon
- Hutao


### 🧠 Long-term Memory System

A lightweight AI Agent memory architecture.

Features:

- Automatic memory extraction
- Memory validation
- Memory consolidation
- Memory retrieval
- Memory importance scoring
- Memory management panel


Memory workflow:

User Input

↓

Memory Extractor

↓

Memory Validator

↓

Memory Manager

↓

Memory Retrieval

↓

Enhanced AI Prompt


### 🎨 Desktop Interaction

- Frameless desktop widget
- Character animation
- Breathing animation
- Talking animation
- Drag interaction
- Chat bubble UI


## 🛠 Tech Stack

| Technology | Usage |
|-|-|
